# de
development environment helper


```
usage: de [<flags>] <command> [<args> ...]

Devenv helper

Flags:
      --help     Show context-sensitive help (also try --help-long and --help-man).
      --all      Apply to all
  -v, --verbose  Verbose output

Commands:
  help [<command>...]
    Show help.

  git pull
    Pull projects and modules

  git status
    Git status on projects and modules

  app start
    Start an app

  app stop
    Stop an app

  npm link
    Npm link modules into projects

  npm install
    Npm install modules

  bower link
    Bower link modules into projects

  bower install
    Bower install modules

  test
    Perform test commands
```
